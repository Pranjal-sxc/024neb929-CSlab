Welcome to the NEB Class 11 Computer Science Syllabus

This webpage provides a comprehensive guide to the Nepal Education Board (NEB) Class 11 Computer Science syllabus. It includes topics related to computer systems, software, hardware, number systems, computer security, programming, and more. This syllabus is designed to equip students with the knowledge and skills required for understanding the fundamentals of computing and digital technologies.

About the New Syllabus

The updated NEB Class 11 Computer Science syllabus offers a structured approach to the study of computer science. It focuses on the following major areas:

  • Introduction to Computer: Basic concepts of computers, history, and evolution.
  • Computer Hardware: The physical components of a computer system, including memory, CPU, and storage devices.
  • Computer Software: Different types of software, including system software, application software, and programming languages.
  • Number System and Boolean Algebra: Understanding number systems, conversion between number systems, and Boolean logic.
  • Computer Security and Cyber Ethics: Cyber threats, privacy concerns, and ethical considerations in the digital world.
  • Introduction to Programming: Basic programming concepts, flowcharts, pseudocode, and control structures.

This syllabus is designed to give students a strong foundation in both theoretical and practical aspects of computer science. It also encourages problem-solving and logical thinking through programming and algorithm design.

NEB Class 12 Computer Science Syllabus

The NEB Class 12 syllabus builds on the foundational knowledge from Class 11 and introduces more advanced topics in programming, databases, networking, and web development.

  • Programming in C: Introduction to C language, syntax, variables, loops, functions, and arrays.
  • Object-Oriented Programming: Concepts of classes, objects, inheritance, encapsulation, and polymorphism (basic overview).
  • Database Management System (DBMS): Introduction to databases, tables, queries, SQL commands (SELECT, INSERT, UPDATE, DELETE).
  • Networking and Data Communication: Types of networks, protocols, network devices, and data transmission concepts.
  • Web Development Basics: HTML, CSS, basic JavaScript concepts for interactive web pages.
  • Project Work: Students are required to complete a mini project involving programming or database/web concepts.
